# attacklens-ai
AttackLens AI
AI Powered Cloud and Identity Attack Path Prioritization Dashboard

Overview
AttackLens AI is a cybersecurity analytics dashboard designed to simulate real world security operations. The application focuses on identifying, analyzing, and prioritizing vulnerabilities across cloud and identity environments using a risk based approach.

It provides clear visibility into attack paths, highlights critical security gaps, and helps security teams focus on remediation efforts that reduce risk effectively.

Features

Risk based prioritization of vulnerabilities using calculated risk scores
Visualization of threat distribution across severity levels
Identification of critical and high risk findings through dynamic alerts
Attack path analysis to represent real world lateral movement scenarios
Priority remediation plan highlighting the most important issues to resolve first
Structured findings including asset details, categories, and mitigation steps

Architecture and Design

The application is built as a client side dashboard using React. Data is processed and visualized to simulate how modern security tools present risk and threat intelligence.

The design follows a modular approach where findings, attack paths, and analytics are handled independently and presented through a unified dashboard interface.

Tech Stack

React JS for front end development
JavaScript for application logic
Recharts for data visualization
CSS for styling and layout

Use Case

This project simulates a scenario where multiple vulnerabilities exist within a cloud environment. For example, an exposed resource can lead to compromised identity access, followed by privilege escalation and full administrative control.

AttackLens AI helps security teams understand this chain of events and prioritize remediation based on actual risk rather than volume of findings.

Key Outcomes

Improved visibility into high risk vulnerabilities
Faster identification of critical issues
Clear guidance on remediation priority
Better understanding of attack path progression
